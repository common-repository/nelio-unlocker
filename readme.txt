== Installation ==

**Installation through the WordPress Dashboard**

1. Login to your WordPress admin.
2. Click on the plugins tab.
3. Click the “Add New” button.
4. Search for “Nelio Unlocker - Importer.”
5. Look for our plugin, click “Install Now” and then “Activate.”
6. Once the plugin is installed and active, you’ll find the conversion page under “Tools” » “Import.” There, look for “Nelio Unlocker” and click on “Run Importer.”


**I successfully installed the plugin. What should I do now?**

First, download the export file of the page you’re interested in using [Nelio Unlocker](https://neliosoftware.com/unlocker/?utm_source=wordpressorg&utm_medium=referral&utm_campaign=nelio-unlocker). Then, in your WordPress site, go to “Tools” » “Import,” look for “Nelio Unlocker,” and click on “Run Importer.” Select the import file you’ve just downloaded and follow the instructions on screen to either create a new page or overwrite an existing one.


== Frequently Asked Questions ==

= How does Nelio Unlocker work? =

[Nelio Unlocker](https://neliosoftware.com/unlocker/?utm_source=wordpressorg&utm_medium=referral&utm_campaign=nelio-unlocker) is an external service that lets you test how a certain page would look like when created using your preferred page builder. After converting any page, you’ll be able to purchase the resulting conversion and download it, so that you can import it into your site.

= Can I edit a converted page? =

Of course. After importing a page, you’ll be able to edit it with your page builder and tweak it as you please.

= Why do I need to create an account in Nelio Unlocker? =

The plugin simply offers a convenient way to import conversions generated by Nelio Unlocker. This means you’ll need to create an account at [Nelio Unlocker](https://neliosoftware.com/unlocker/?utm_source=wordpressorg&utm_medium=referral&utm_campaign=nelio-unlocker) in order to convert your existing pages into your desired page builder.
